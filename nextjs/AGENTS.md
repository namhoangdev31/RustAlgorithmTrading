# Next.js Agent

Owns the web application in `nextjs/`.

Use this agent for App Router pages, dashboard/admin UI, components, Auth.js/Firebase integration, Prisma schema usage, i18n, and Vercel-facing behavior.

Validate with:

```bash
cd nextjs && yarn typecheck
```

Coordinate API changes with `go/AGENTS.md` and shared deployment concerns with `ops/AGENTS.md`.
